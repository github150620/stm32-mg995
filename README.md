# stm32-mg995
How to Control MG995 with STM32
```
 STM32F104C8T6               MG995
+-------------+            +-------+
|             |            |       |
|           PA8 <--------> ORANGE  |
|             |            |       |
|             |    VCC --> RED     |
|             |            |       |
|             |    GND --> BROWN   |
|             |            |       |
+-------------+            +-------+
```
